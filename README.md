# basta-game

A classic "Basta" (also known as "Stop" or "Scattergories") word game implemented as a web application. 

Players choose a letter and a category, then race against the timer to write words that start with that letter. It is a fun, interactive project designed to test DOM manipulation and rapid state updates.

**Tech Stack:** HTML, CSS, JavaScript.
**Key Features:** Timer, dynamic category selection, and real-time scoring.
